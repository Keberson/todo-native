## Overview

### Technical specification

Create ToDo application:

Application must contain:
- Task list
  - View
  - Find
  - Create
- Task screen
  - View
  - Edit
  - Delete

The task consist of: 
- name
- description
- priority
- is done

ToDo App must use offline storage.
